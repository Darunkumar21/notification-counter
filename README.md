# 🔔 Notification Counter

Cross-platform notification counter that works on **Desktop, Mobile, Tablet** and any device!

## ✨ Features
- ✅ **Works everywhere** (PWA technology)
- ✅ **Real notifications** with sound/vibration
- ✅ **Persistent storage** (survives browser close)
- ✅ **Today/Total counters**
- ✅ **Installable as app**
- ✅ **Offline capable**

## 🚀 Quick Start
1. Clone/Download
2. Open `index.html` in any browser
3. Click "Test Notification" 
4. Install as PWA for best experience!

## 📱 Platforms Supported
- Desktop (Chrome, Firefox, Safari, Edge)
- Mobile (iOS Safari, Android Chrome)
- Tablets

## 🔧 Customization
Edit `script.js` to:
- Change increment triggers
- Add auto-increment intervals
- Integrate with APIs
